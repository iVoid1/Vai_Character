# Vai Character
a small tool to make your character alive and see it express it's emotion
---
## VMC
  An easy way to use VMC interface with smooth transitions between expressions

## Agent
  Using langchain ollama to make everything locally and very fast
  you can give it some tools to give the character some control

## Memory system
  Sqlite database for saving the most emotional messages in the session so you can continue you chat with the same subject 
  useing embedding model to improve accuracy of what should be send to the model saving you a huge amount of tokens
